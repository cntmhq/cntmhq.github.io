═══════════════════════════════════════════════════════════════════════════════
                          ANIMATION MATHEMATICS
═══════════════════════════════════════════════════════════════════════════════

Linear Interpolation (Lerp):
  result = start + (end - start) * t

  Example with t ∈ [0, 1]:
    lerp(3.5, 4.0, 0.5) = 3.5 + (4.0 - 3.5) * 0.5
                        = 3.5 + 0.25
                        = 3.75 ✓

Easing Function (Cubic Ease-In-Out):
  Transforms linear t to smoother curve

  t < 0.5:  eased = 4 * t³
  t ≥ 0.5:  eased = (t - 1) * (2t - 2)² + 1

  Result: Smooth S-curve acceleration
